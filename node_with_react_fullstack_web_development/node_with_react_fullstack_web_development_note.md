# Node with React: Fullstack Web Development

## 1. git & heroku 명령어
```
git status
git add .
git commit -m "message"
git push heroku master
git remote add heroku https://git.heroku.com/{heroku url}.git

heroku -v
heroku login
heroku create
heroku help
heroku apps:destroy {app-name}
heroku open
heroku logs
```
